Lupa Kata Sandi?

Ikuti langkah berikut untuk memulihkan akses Anda

Untuk keamanan data, pemulihan kata sandi hanya dapat diproses oleh TIM INFORMASI sistem.

1

Siapkan data diri resmi Anda (NIP atau Nama Pengguna).

2

Hubungi TIM INFORMASI melalui email: admin@dprd.jatimprov.go.id untuk verifikasi kepemilikan akun.

3

Kata sandi baru akan diberikan secara aman setelah proses validasi selesai.
